# FlappyBirdDemo-VPython

This is a demo of FlappyBird game. Written in Python and uses VPython v6 library for rendering.

The VPython library can be download at:
http://vpython.org/contents/download_windows.html (windows)
http://vpython.org/contents/download_mac.html (mac os)
http://vpython.org/contents/download_linux.html (linux)
